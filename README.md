# IconicGreen LLC site

Minimal static website for IconicGreen LLC.

## Pages

- `index.html` - Company website and Client URI
- `privacy.html` - Privacy Policy URI
- `terms.html` - Terms of Service URI

## Before publishing

Replace `hello@iconicgreen.example` with the real company email address.

## Quick deploy options

- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages
